# Dia 02 - Analisador de logs

Cria um script que le um ficheiro de logs e gera um resumo.

## Requisitos

- Ler um ficheiro `.log` ou `.txt`.
- Contar linhas por nivel: `INFO`, `WARNING`, `ERROR`, `DEBUG`.
- Mostrar os 5 erros mais frequentes.
- Guardar um resumo em `relatorio.txt`.

## Extra

Aceita o caminho do ficheiro por argumento de linha de comando.

## Exemplo de linha

```text
2026-01-10 12:30:22 ERROR Falha ao conectar ao banco
```
